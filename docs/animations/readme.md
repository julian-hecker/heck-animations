## Animations
To use an animation, use the following class: `ha-`*name*, where *name* is the name of an animation from the list below.

Animations are classified into the following categories: 
- Slides
- Fades
- Rotates
- Scales

### Slides
- Slide-top
- Slide-right
- Slide-bottom
- Slide-left
